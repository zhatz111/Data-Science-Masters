Run the Jupyter Notebook for Data Exploration follow these steps:
	1. Ensure a valid python installation is present on your computer
	2. Install the packages from the requirements.txt file using "pip install -r requirements.txt"
	3. Ensure that the CSV data file is in the same directory as the Jupyter Notebook
	4. At the top of the file in VS Code hit "Run All" to run all the code


For the Power BI Dashboard:
	1. Open the .pbix file
	2. The data should be stored within the file itself so no need to search for it
	3. The data source file is in this zipped folder if needed
